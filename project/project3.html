Analisis komparatif antara algoritma Support Vector Machine dan Convolutional Neural Network untuk memprediksi kelayakan kredit berdasarkan data historis

Informasi Proyek

Penelitian ini membandingkan performa dua algoritma machine learning yang berbeda - Support Vector Machine (SVM) dan Convolutional Neural Network (CNN) - dalam memprediksi kelayakan peminjaman berdasarkan data historis dari 614 aplikasi pinjaman.

Latar Belakang

Penilaian kelayakan kredit merupakan proses penting dalam industri keuangan untuk meminimalkan risiko kredit macet. Penelitian ini membandingkan dua pendekatan machine learning yang berbeda: SVM sebagai algoritma tradisional yang powerful untuk klasifikasi, dan CNN sebagai representasi deep learning yang dapat menangkap pola kompleks dalam data.

Sumber Data

Dataset penelitian bersumber dari kompetisi Loan Prediction di platform Analytics Vidhya:

  • Dataset asli: 614 sampel dengan 13 fitur
  • Variabel meliputi: informasi demografis (gender, status pernikahan, tanggungan), pendidikan, status pekerjaan, pendapatan, jumlah pinjaman, jangka waktu pinjaman, riwayat kredit, dan lokasi properti
  • Target biner: Loan_Status (Disetujui/Ditolak)

Metodologi

Penelitian menggunakan pendekatan berikut:

  • Praproses data: handling missing values, encoding kategorikal, feature scaling
  • Pembagian data: 80% training, 20% testing dengan stratifikasi
  • Pemodelan dengan SVM (dengan GridSearchCV) dan CNN
  • Evaluasi dengan accuracy, precision, recall, F1-score, dan confusion matrix
  • Analisis distribusi probabilitas prediksi

Preprocessing Data

Beberapa teknik preprocessing yang diterapkan:

  • Penanganan missing values dengan modus untuk kategorikal dan median untuk numerik
  • Label encoding untuk variabel kategorikal
  • StandardScaler untuk normalisasi fitur numerik
  • Reshape data untuk input CNN (dari 11 fitur menjadi 11x1)

Perbandingan Metode

Analisis komparatif antara algoritma SVM dan CNN

Aspek Support Vector Machine (SVM) Convolutional Neural Network (CNN)
Prinsip Algoritma Mencari hyperplane optimal untuk memisahkan kelas Jaringan saraf tiruan dengan lapisan konvolusi untuk ekstraksi fitur
Hyperparameter Tuning GridSearchCV: C=1, kernel='rbf', gamma=0.01, class_weight='balanced' Arsitektur: 2 layer Conv1D (64 dan 32 filter), 3 layer Dense (128, 64, 32 neuron)
Data Input Vektor fitur tradisional (11 dimensi) Data berbentuk sequence (11x1) untuk lapisan konvolusi
Waktu Training Relatif lebih cepat (dengan GridSearch) Relatif lebih lambat (100 epochs dengan early stopping)
Kemampuan Generalisasi Baik untuk dataset kecil, kurang rentan overfitting Memerlukan lebih banyak data, potensi overfitting tanpa regulasi

Hasil Penelitian

Metrik evaluasi dan hasil prediksi model machine learning yang dikembangkan

Accuracy: 86.18%

SVM mencapai akurasi tertinggi pada dataset testing

Recall: 98.82%

SVM sangat baik mengidentifikasi pinjaman yang layak disetujui

F1-Score: 90.81%

Nilai F1-score menunjukkan keseimbangan precision-recall yang baik

Class Imbalance

Dataset tidak seimbang (337 disetujui vs 154 ditolak)

Kesimpulan

Berdasarkan hasil penelitian, SVM dengan parameter terbaik (C=1, kernel='rbf', gamma=0.01, class_weight='balanced') menghasilkan performa yang sangat baik dengan accuracy 86.18%, precision 84.00%, recall 98.82%, dan F1-score 90.81%. Model ini sangat efektif dalam mengidentifikasi pinjaman yang layak disetujui (recall tinggi) namun kurang optimal dalam mendeteksi pinjaman berisiko (precision rendah untuk class ditolak).

CNN menunjukkan potensi yang baik tetapi memerlukan lebih banyak data dan tuning yang lebih mendalam untuk mengungguli SVM pada dataset yang relatif kecil ini. Pemilihan algoritma tergantung pada prioritas bisnis: jika tujuan adalah meminimalkan penolakan terhadap peminjam yang layak (false negative), SVM dengan konfigurasi ini sangat direkomendasikan.
