Detail Proyek

Penjelasan mendalam tentang latar belakang, data, dan metodologi yang digunakan dalam proyek

Latar Belakang

Proyek ini bertujuan membantu petani mengurangi penggunaan pupuk nitrogen sintetis dengan memprediksi curah hujan. Air hujan membantu proses leaching yang menguraikan nitrogen berlebih di tanah. Prediksi yang akurat memungkinkan petani mengoptimalkan irigasi dan mengurangi dampak negatif pupuk terhadap lingkungan.

Sumber Data

Dataset diperoleh dari Agreed Earth LTD:

  • Data curah hujan harian Kota Devon (2010-2012)
  • Total 3 tahun data dengan 52 data points per tahun
  • Data dikumpulkan dari stasiun cuaca di wilayah Devon
  • Data telah melalui proses quality control dan cleaning

Preprocessing Data

Beberapa teknik preprocessing yang diterapkan:

  • Aggregasi data harian menjadi data mingguan
  • Transformasi data menggunakan Min-Max Scaler (0-1)
  • Penentuan variabel input berdasarkan analisis ACF dan PACF
  • Pembagian data: 80% training, 20% testing
  • Handling missing values dengan interpolasi

Variabel Input

Berdasarkan analisis ACF dan PACF, variabel input yang digunakan:

  • Lag-1, Lag-2, Lag-3, Lag-5, Lag-6, dan Lag-15
  • Variabel target: data curah hujan yang telah dilakukan differencing 1 kali
  • Feature engineering: moving averages dan seasonal indicators

Perbandingan Model Machine Learning

Analisis perbandingan berbagai algoritma machine learning yang diimplementasikan

Model Prinsip Algoritma Keunggulan Kelemahan
Regresi Linear Memodelkan hubungan linear antara variabel input dan target Interpretasi mudah, cepat dalam training Asumsi linearitas, kurang cocok untuk pola kompleks
Random Forest Ensemble method dengan multiple decision trees (bagging) Reduksi overfitting, handle missing values Waktu training lebih lama, kurang interpretable
XGBoost Ensemble method dengan gradient boosting Akurasi tinggi, regularization untuk hindari overfitting Hyperparameter tuning kompleks, computational cost tinggi
LSTM Jaringan saraf tiruan recurrent untuk data sequential Tangkap pola temporal jangka panjang, handle non-linearitas Data intensive, computational cost sangat tinggi

Kesimpulan

Berdasarkan hasil penelitian, Regresi Linear menunjukkan performa terbaik dalam memprediksi curah hujan di Kota Devon dengan nilai error terendah (RMSE: 4.454, MAE: 3.158). Hasil prediksi menunjukkan curah hujan minggu pertama tahun 2013 sebesar 3.399 mm yang tergolong rendah, sehingga petani perlu menyiapkan air tambahan untuk mengurangi kandungan nitrogen pada tanah.

Meskipun model yang lebih kompleks seperti XGBoost dan LSTM biasanya diharapkan memberikan hasil yang lebih baik, dalam kasus ini Regresi Linear cukup untuk menangkap pola data curah hujan yang relatif stabil. Hal ini menunjukkan bahwa untuk dataset dengan karakteristik tertentu, model yang lebih sederhana dapat memberikan hasil yang optimal dengan computational cost yang lebih rendah.
